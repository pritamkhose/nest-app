## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# run postman collection
$ npm run postman

# run postman collection
$ npm run docs
```

## nest cli
- [Nest Cli Documentation](https://docs.nestjs.com/recipes/crud-generator)
```bash
$ nest g resource users
```
